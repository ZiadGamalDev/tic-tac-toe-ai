# Tic Tac Toe AI

A simple AI-based Tic Tac Toe game built with Python and Tkinter.

## Features

- Single and multiplayer modes
- Minimax algorithm for unbeatable AI
- Easy-to-use GUI

## How to Run

```bash
python Tic-Tac-Toe.py
```

Make sure you have Python installed.
